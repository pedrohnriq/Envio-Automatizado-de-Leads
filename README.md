# Automação de Leads

Projeto desenvolvido em Python para automatizar a coleta, processamento e envio de informações de leads.

## Tecnologias

- Python
- Pandas
- SQL Server
- SQL
- SMTP
- Shell Script

## Estrutura

```
main.py           -> Arquivo principal
conexao.py        -> Conexão com banco
query.sql         -> Consulta SQL
l_email.py        -> Envio de e-mails
e_leads_edt.py    -> Tratamento dos dados
```

## Como executar

```bash
pip install -r requirements.txt
python main.py
```
